Descarga e instalacion

El SDK está disponible en Steam para los propietarios de Counter-Strike: Global Offensive con Prime Status con Windows. El Prime Status debe comprarse a través de CS: GO o Steam, o habiendo comprado CS: GO antes de que se vuelva gratis para jugar.

  1. Abre la ventana de Steam
  2. Haga clic en la pestaña BIBLIOTECA y luego activa opcion herramientas. Luego de eso, parecera abajo de CSGO.
  3. Una vez que le de a instalar, el SDK de CSGO se descargará en su computadora a través de Steam.
  4. Una vez que se haya instalado por completo, podrá ejecutarlo

Este SDK incluye:

  • Hammer World Editor, el editor de nivel de Source Engine
  • Conjunto de utilidades de compilación/li>
  • Face Poser actualizado
  • Mapas de ejemplo y prefabricados
